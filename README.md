# VK_Bot
Бот для соц.сети vk.com откликающийся на имя Антошка, предназначеный, в основном, для внутренних нужд беседы учебной группы (для лулзов в беседе учебной группы, если быть точным)  
### Некоторые возможности:
* Добавление в базу картинок и видео с мемасиками и последующий вброс их в беседу по команде
* Поиск значения слова в википедии
* Выбор варианта из предложеного списка
* Похвальба или ругань кого-либо из беседы по требованию
* Не документированые возможности (отсутствуют в предоставленнной версии кода)
